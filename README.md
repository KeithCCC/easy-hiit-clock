# Easy HIIT Clock

A simple HIIT (High-Intensity Interval Training) timer with visual exercise guide.

## Features
- Large, easy-to-read timer display
- Exercise indicator showing current workout step
- GO/STOP controls for manual progression
- Progress bar for each exercise
- Audio beep on exercise transitions
- Responsive design

## Usage
Open `index.html` in your browser or deploy to Vercel for online access.

Press **GO** to start each exercise timer, **STOP** to pause, and **RESET** to start over.

## Deployment
This app is ready to deploy on Vercel.
